# Install
```
git clone https://github.com/emibcn/react-test-graph-app.git
cd react-test-graph-app
```

# All in one
```
docker-compose up --build -d; docker-compose logs -f
```

# Build
```
docker-compose build
```

# Run
## Background
```
docker-compose up -d
```

## Foreground
```
docker-compose up
```

# Logs (with follow)
```
docker-compose logs -f
```

# Stop
```
docker-compose down
```
